Import the data:
1. Indicate the table Id
2. Check "Upload" and "Insert"
3. Indicate the file name
4. If you haven't got yet the XML-file with data, pres the button "Create XML File". It creates the template-file.
4.1 Fill-in the file in excel and save it as XML.
5. Press the button "Upload".

Update the data.
1. Indicate the table Id
2. Indicate the Index ID. It will be used for searching the record for update. Index should be unique.
3. Check "Upload" and "Update"
4. Indicate the file name
5. Press the button "Update the list of fields".
6. On the tab "Update option" for fields which should be updated check the field "Update"
7. Press the button "Upload"

Export whole table
1. Indicate the table Id
2. Indicate the file name
3. Press the button "Export table"
